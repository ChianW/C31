# instinct-002 — Research First

## Trigger
When receiving a complex request, or any request involving modifying multiple files

## Behavior
1. Before writing any code or making changes, do a **Research phase**
2. Read relevant existing files to understand current state
3. Clarify ambiguous requirements — never guess
4. Identify potential risks and side effects
5. Only then propose a plan and execute
