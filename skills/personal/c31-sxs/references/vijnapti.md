# 四寻思观的唯识宗背景参考

## 四寻思观（Catur-pravicāra）

出自《瑜伽师地论》和《成唯识论》。是加行位（准备阶段）的核心修行方法，目的是通过逻辑解构，瓦解"遍计所执"（强加于事物的虚构概念）。

四寻思的对象：
- 名（nama）→ 语言标签
- 事（artha）→ 被指称的对象/事件
- 自性（sva-lakṣaṇa）→ 被认为固定不变的本质
- 差别（viśeṣa）→ 价值判断、比较、优劣

## 三性（Trisvabhāva）框架

四寻思观最终要让人认识到：
- **遍计所执性**：我们强加于世界的虚构概念（如"我失败了"）→ 完全为空
- **依他起性**：因缘和合的流动现象（如"现金流归零"作为事件）→ 有，但不是本质
- **圆成实性**：去掉虚构概念后，事物如其所是的真实状态 → 真实

## 与 Hoffman ITP 的对应

| 唯识宗 | Hoffman ITP |
|--------|-------------|
| 遍计所执（名、差别） | 桌面图标（用户强加的标签） |
| 依他起（事、因缘） | 界面背后的交互数据 |
| 圆成实 | 不可直接被界面访问的底层真实 |
| 末那识（执我） | 自我图标（self-icon） |
| 阿赖耶识（种子） | 适应度优化编码的底层数据结构 |

## 末那识的运作

末那识（kliṣṭa-manas）恒常地审察思量，把阿赖耶识的见分（认识能力）执为"我"。它不是在抓前六识，而是抓第八识的能动性本身，然后让前六识的所有认知都带着"与我相关"的滤镜。

这就是"我执"的机械原理。

## 四如实智（Four Correct Knowledges）

四寻思的完成态：
1. 名如实智：知道名称只是约定俗成，不对应本质
2. 事如实智：知道事件只是因缘和合，没有独立自性
3. 自性如实智：知道没有固定不变的本质
4. 差别如实智：知道价值判断是分别心造作，不是客观属性

## 实用提示

- 四寻思不是"想通"，而是"拆解"。
- 不需要相信佛教，只需要愿意把"我觉得真的"当成"需要被检查的对象"。
- 最常被卡住的地方通常是差别寻思——因为价值判断往往是早年植入的，用户会把它当成"我的真实感受"而不愿放手。
